I Don't Give A Fuck Public License

IDGAFPL which stands for I Don't Give A Fuck Public License is a Free Software license which you can easily use in your Free, Libre and Open Source projects.

How to use this license in a project

Just copy and paste the following phrases in a file called LICENSE in your project's directory: 

                I Don't Give A Fuck Public License YEAR (c) COPYRIGHT_HOLDER 

                I DON'T GIVE A FUCK ABOUT WHAT YOU DO WITH THIS CODE.

Copyleft version

As you may be a GNU GPL fan (I have no fucking idea why you just don't use the original one), we also provided a copyleft compatible version for you: 

                I Don't Give A Fuck Public License YEAR (c) COPYRIGHT_HOLDER

                I DON'T GIVE A FUCK ABOUT WHAT YOU DO WITH THIS CODE UNLESS YOU SHARE WHAT YOU DID UNDER THE SAME LICENSE.
